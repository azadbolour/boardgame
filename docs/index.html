Board Game

A board game in the crossword genre.

Showcasing Haskell and Scala game servers and a React drag and drop web UI.

Currently in version 0.9 alpha trial. For modern browsers on the MAC, Linux, and Windows platforms. Not supported on smart phones and tablets.

Check it out!

Drag and drop letters to form a word connected to an existing board letter (if any) and click Commit Play to play that word.

The application represents the most recent deployment of the boardgame open source project. The project is intended primarily as a test bed for learning and experimentation with different languages and framework for application development and deployment.

The game's rules may change over time as interesting variations are discovered with usage.

Rules

  • Points are associated with board locations - not with letters.
  • The assignment of points to board locations varies from game to game.
  • A word play accumulates the points of the [new] locations it captures.
  • A word play on a non-empty board must contain at least one existing letter.
  • A word play on an empty board is unrestricted.
  • The played word and any crosswords formed by the play must exist in the game's dictionary.
  • The machine generates letters for play randomly based on the frequency of letters in English lexicon (per Oxford Math Center). There is no notion of a container of letters getting depleted.
  • When no play is possible, a single letter is swapped for another. A game ends upon 10 consecutive no-plays [swaps].
  • Games have a time limit of 1 hour. Longer games are considered abandoned and are harvested.

Noteworthy

You may wish to install a dictionary plugin in your browser to easily search for the meanings of words played by the machine.

Make sure the status field below the board is visible.

The starting player is chosen at random to be the user or the machine. There may be a slight hesitation in an initial machine play.

The board's dimension is configurable through the URL query parameter dimension. Valid values are odd numbers between 5 and 25 inclusive (default 17).

The game's letter tray capacity is configurable through the URL query parameter tray-capacity (in the range [3 .. 15], default 8).

Issues

The game's dictionary is based on:

http://www.gutenberg.org/files/3201/files/CROSSWD.TXT -

a comprehenve list of common English words.

But from time to time, you'll run into a common word that is not in the dictionary.

Please add issues and ideas for improvment (including any missing common words in the dictionary) to the issues list.
